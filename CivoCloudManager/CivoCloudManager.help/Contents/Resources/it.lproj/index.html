Aiuto Civo Cloud Manager

Gestisci la tua infrastruttura Civo Cloud dal tuo Mac.

Per iniziare

Civo Cloud Manager funziona come app nella barra dei menu. Dopo l'avvio, un'icona a forma di scudo appare nella barra dei menu del tuo Mac.

  1. Fai clic sull'icona dello scudo per aprire il popover.
  2. Al primo avvio, la Configurazione guidata si apre automaticamente.
  3. Inserisci la tua Civo API Key (disponibile su civo.com → Account → Security → API Keys).
  4. Seleziona la tua regione (es. fra1, lon1, nyc1).
  5. Scegli quali firewall gestire e configura le porte.
  6. Facoltativamente, abilita Avvia al login.

Barra dei menu

Il popover della barra dei menu offre una gestione rapida dei firewall:

IconaSignificato
Scudo (verde)Tutti i firewall chiusi
Scudo (giallo)Alcuni firewall aperti per il tuo IP
Scudo (rosso)Configurazione non completata
  • Apri — crea una regola firewall che consente il tuo IP pubblico attuale sulla porta configurata.
  • Chiudi — rimuove la regola.
  • Apri tutti / Chiudi tutti — azioni di massa per tutti i firewall gestiti.
  • Dashboard — apre la finestra di gestione completa.
  • Impostazioni — riapre la configurazione guidata.
  • Aggiorna — verifica manualmente lo stato.

Il tuo IP pubblico viene rilevato automaticamente tramite ipify.org con provider di fallback. L'app gestisce solo le regole che ha creato — contrassegnate con il tuo hostname.

Dashboard

Il Dashboard mostra il conteggio delle risorse e l'utilizzo delle quote. Fai clic su qualsiasi scheda risorsa per navigare direttamente a quella sezione.

Indicatori di quota

Gli indicatori circolari mostrano l'utilizzo rispetto al limite per tutte le quote dell'account (RAM e DB RAM visualizzati in GB).

Richiedi modifica quota

Fai clic su Richiedi modifica accanto a «Utilizzo quota» per aprire un modulo con stepper per tutti i limiti di quota. Invia una richiesta di modifica tramite la Civo API.

Compute

Istanze

Visualizza, crea ed elimina istanze di calcolo. Fai clic su un'istanza per vedere i dettagli:

  • Specifiche CPU, RAM e disco
  • Indirizzi IP pubblici e privati
  • Password root iniziale (pulsante mostra/nascondi)
  • Tag, regione, data di creazione
  • Associazioni di rete e firewall

Per creare: fai clic su + nella barra degli strumenti. Compila hostname, dimensione, immagine disco, rete, firewall, SSH Key e tag.

SSH Keys

Gestisci le SSH Keys per l'accesso alle istanze. Per aggiungere una chiave: fai clic su +, inserisci un nome e incolla la tua chiave pubblica.

Kubernetes

Cluster

Visualizza, crea ed elimina cluster Kubernetes. Fai clic su un cluster per vedere la vista dettagliata con:

  • Statistiche — Nodes, Pools, stato di salute, applicazioni installate a colpo d'occhio.
  • Info cluster — endpoint API, IP master, DNS, plugin CNI, dimensione node, data di creazione.
  • Condizioni — ControlPlaneReady, WorkerNodesReady, ClusterVersionSync con indicatori di salute.
  • Node Pools — fai clic per espandere e vedere ID pool, dimensione, numero di nodes, IP pubblici, nomi istanze, labels e taints.
  • Labels — fai clic su Modifica per aggiungere, rimuovere o modificare le labels di un node pool.
  • Applicazioni installate — cert-manager, ingress-nginx, ecc.

Per creare: fai clic su +. Configura nome, plugin CNI, dimensione node, numero di nodes, rete e app del marketplace.

Rete

Reti

Visualizza e crea reti virtuali. La rete predefinita non può essere eliminata. Fai clic su + per crearne una con un'etichetta e CIDR opzionale.

Firewall

Fai clic su un firewall per vedere tutte le sue regole. Ogni regola mostra:

  • Protocollo, intervallo porte, CIDR
  • Direzione (ingress/egress) e azione (consenti/nega) come badge colorati
  • Etichetta

Fai clic su + nella vista dettagliata delle regole per aggiungere una nuova regola. Elimina le regole tramite il menu contestuale (clic destro, richiede la digitazione del nome della regola).

Load Balancer

Visualizza i dettagli del load balancer, inclusi algoritmo, IP pubblici/privati, politica di traffico e lista dei backend. Eliminazione tramite menu contestuale.

Domini & DNS

Espandi un dominio per vedere tutti i record DNS. Ogni record mostra tipo, nome, valore e TTL.

  • Aggiungi record — fai clic sul pulsante «Aggiungi record» all'interno del dominio.
  • Modifica record — fai clic destro su un record e seleziona «Modifica».
  • Elimina record — fai clic destro e seleziona «Elimina».

Archiviazione & dati

Database

Fai clic su un database per vedere i dettagli di connessione (host, porta, voce DNS, IP privato), configurazione (software, versione, dimensione, nodes) e associazioni rete/firewall.

Per creare: fai clic su +. Scegli PostgreSQL o MySQL, versione, dimensione, nodes, rete e firewall.

Volumi

Fai clic su un volume per vedere lo stato di collegamento (a quale istanza o cluster è collegato), punto di montaggio, dimensione, stato di avvio, regione e data di creazione.

La lista mostra se ogni volume è «collegato a un'istanza», «collegato a un cluster» o «disponibile».

Pulisci disponibili — pulsante nella barra degli strumenti per eliminare in massa tutti i volumi con stato «disponibile». Richiede la digitazione di DELETE ALL DATA per confermare.

Object Store

Visualizza gli object store con endpoint e dimensione massima. Fai clic destro per:

  • Mostra credenziali — mostra l'access key ID e il secret access key (selezionabile per copiare).
  • Elimina — rimuove permanentemente l'object store.

Account

Regioni

Visualizza tutte le regioni Civo disponibili. Fai clic su Usa per cambiare la tua regione attiva. Tutte le successive chiamate API saranno dirette alla regione selezionata.

Informazioni

Mostra la versione dell'app, il numero di build, le informazioni sullo sviluppatore e i link all'Informativa sulla privacy e ai Termini di utilizzo.

Eliminare risorse

Tutte le operazioni di eliminazione richiedono la digitazione del nome esatto della risorsa per confermare. Questo previene l'eliminazione accidentale di infrastruttura cloud critica.

Per eliminare: fai clic destro su una risorsa e seleziona Elimina. Appare una finestra di conferma in cui devi digitare il nome esattamente come mostrato. Il pulsante elimina si attiva solo quando il nome corrisponde.

Accesso completo

La gestione dei firewall nella barra dei menu è gratuita. Il dashboard completo con tutta la gestione delle risorse richiede un acquisto una tantum di Accesso completo (14,99 $).

  • Ripristina acquisto — se hai acquistato precedentemente su un altro dispositivo.
  • Riscatta codice — per codici promozionali Apple.
  • Condivisione familiare — abilitata.

Suggerimenti

  • Tutti i campi di testo con IP, endpoint e credenziali supportano la selezione del testo per una facile copia.
  • L'app aggiorna automaticamente lo stato dei firewall ogni 60 secondi.
  • Le schede risorse del dashboard sono cliccabili — naviga in qualsiasi sezione con un clic.
  • Le regole firewall create da questa app sono etichettate come civo-cloud-<hostname>-<firewall> e vengono gestite solo le regole di questa app.
